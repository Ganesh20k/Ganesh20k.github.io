My Project

Introduction

Welcome to my portfolio of project, where I showcase a range of web applications primarily focused on healthcare management, video communication, and pharmacy systems. All the applications in one complete website it highlights my skills in full stack development using the Spring Boot framwork, Thymeleaf templating, and MySQL database integration, along with modern frontend technologies like HTML, CSS, and JavaScript.

Healthcare Management System

A comprehensive application designed to streamline hospital and clinic operations. Features include doctor and patient managemant, appointment scheduling, and secure authentication. Built with a Spring Boot backend and Thymeleaf frontend, it ensures seamless interaction and data integrity.

Pharmacy Management System

An efficient system for managing multiple pharmacies, enabling easy medicine stock updates, order routing to nearby pharmacies, and detailed inventory controls. Integrated locations for fast delivery.

Online Doctor Consultation

A video-enabled consultation feature connecting patients with doctors via secure, real-time video calls.The system integrates Spring Boot backend services with frontend video APIs to deliver a smooth, interactive user experience.

Medical Checkup and Nursing Booking

A user-friendly application allowing customers to book home medical checkups and nursing services. Includes features like booking management, payment processing with assign a nurse and send a whatsApp notification to the Assigned nurse.

Diagnosis Lab Management System

A dedicated platform facilitating the management of diagnostic labs, enabling lab test requests, result entry, and report management. This system integrates seamlessly with patient and doctor modules, providing lab test tracking and status updates for efficient medical workflows.
